# Projet Flask - Formulaire d'Inscription

## Prérequis

- Python 3.x
- Flask

## Installation

1. Créez un environnement virtuel et activez-le :
   ```bash
   python -m venv venv
   source venv/bin/activate  # Sur Windows, utilisez `venv\Scripts\activate`
   ```

2. Installez Flask :
   ```bash
   pip install Flask
   ```

## Structure du Projet

- `app.py`
- `templates/`
  - `index.html`
  - `form.html`
  - `confirmation.html`
- `static/`
  - `styles.css`
- `venv/`

## Lancement de l'application

```bash
python app.py
```
